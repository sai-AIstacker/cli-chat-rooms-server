# Command Line Chat Tool

A simple multi-room chat application built with Python using socket programming and threading.

## Features
- Multiple chat rooms support
- Create and join chat rooms
- Real-time messaging
- Username support
- Simple command-line interface

![alt text](image-1.png)


## Requirements
- Python 3.6 or higher

## How to Run

### Start the Server
```bash
python server.py
```

### Connect Clients
Open new terminal windows and run:
```bash
python client.py
```

## Usage
1. Enter your username
2. Choose to list, create, or join a chat room
3. Start chatting!
4. Type `/quit` to leave

## Project Structure
- `server.py` - Main server that handles all connections and rooms
- `client.py` - Client program for users to connect and chat
